Zombie
Dash

* Mintbean Star Award Winner *

Run from zombies. Dodge fireballs. Nuke the undead.

  • Live Site
  • Github Code

Zombie Dash was built in 7 days by a team of 3 developers for the August 2021 Mintbean Hackathon. The challenge was to build a 2-D game, making the Phaser JS framework the perfect choice.

Zombie Dash received the Mintbean Star Award, an honor given to only 5% of developers. Recipients are listed individually at hire.mintbean.io.


This group project was created with Phaser JS.
